# Chat Server

A Chat Room Web Application (Go + gRPC + Angular6)

## Environment
Ensure you have setup the environment for [Go](https://golang.org/doc/install), [gRPC](https://grpc.io/docs/quickstart/go.html)
```
$ echo $GOPATH
(HOME DIR)/go
```

## Build
```
$ make build
```

## Run
### server process
```
$ messagingserver
```

### client process
```
$ messagingclient
type some input and see what happens
```
